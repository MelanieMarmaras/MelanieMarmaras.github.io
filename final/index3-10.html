PROCESS BOOKLETS

process of past class assignments // may 2014

  • make paper, laser cut, saddle stitch
  • client typography II class
  • discipline publication, typography
  • position designer, production

These are a set of typographic booklets showing process for each assignment of that class. For each assignment I had reach realization points or epiphanies where I made break troughs, which are laid out at different points in each booklet. Each booklet shows strides over the coarse of the project and that theme influenced my type choices, paper stock, color, and laser cut choices as elements I had not experimented with in previous assignments.
